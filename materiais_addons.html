Imports images as Planes

Permite importar rapidamente imagens mapeadas a um plano (cria um plano para cada imagem).

Cria uma nova opção no menu de Import (File>Import>Images as Planes) que permite importar imagens directamente aplicadas/mapeadas a planos. Pode seleccionar e importar várias imagens, com ou sem transparência, de uma só vez. O Blender cria um plano para cada uma das imagens importadas.

Este addon já vem com o Blender, basta ativar.
