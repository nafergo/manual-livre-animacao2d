Sheep it! render farm

O Sheep it! é um serviço online gratuito de renderização distribuída para ficheiros criados com Blender. Por outras palavras, é uma renderfarm online gratuita para Blender.

Renderizar os meus projetos

Para renderizar os seus projeto tem de começar por criar uma conta no Sheep It!.

Depois, visite a página Get Started para enviar o seu ficheiro.

    Respeite as regras do serviço:
  • Só é possível renderizar em Blender Internal e Cycles
  • O output deverá ser JPG ou PNG
  • Os caminhos para "external data" (caso exista) devem ser relative
  • Na Performance, o número de threads deverá estar definido para Auto-detect.
  • Os ficheiros não devem usar acentos, ç ou espaços em branco.
  • O tempo de renderização deverá ser inferior a 20min por frame num pc quad-core
  • Deve limitar a renderização a 3000 frames por ficheiro
  • Pode enviar ficheiro .blend (com external data packed) ou ficheiro zip mas tenha atenção aos caminhos (devem ser relativos!)

Depois, é ir verificando o progresso, fazer download das frames quando estiver concluída a sua renderização e remover o projeto depois de terminado.

Renderizar os projetos de outros?!?

O Sheep it! só funciona porque existem pessoas no mundo inteiro que disponibilizam processamento para renderizar os projetos de outros. Não existe reciprocidade obrigatória: pode só enviar os seus projetos para renderizar ou só renderizar projetos de outros. No entanto, o mais comum é as duas atividades estarem associadas.

Ao disponibilizar o processamento do seu computador e renderizar frames de projetos de outro, ganha créditos. Estes créditos são depois utilizados para estabelecer a prioridade nos projetos a renderizar. Ou seja, quantos mais créditos tiver mais curto é o seu tempo na lista de espera.

Se tiver computador(es) seu(s) ligados ao Sheep It! e enviar um projeto, o seu projeto tem prioridade no(s) seu(s) computador(es). A utilização do restantes computadores conectados depende da sua prioridade na lista de espera e a sua posição é determinada pelos créditos.

Para conectar o seu computador tem de ter Java instalado e pode escolher entre duas opções:

  • Pode abrir o browser e visitar a página Get Started para iniciar o processo.
  • Se não quiser abrir o browser, pode ativar o cliente Java através da linha de comando. Encontra as instruções na mesma página.
