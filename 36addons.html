Add-Ons expandindo as funcionalidades originais

Uma das grandes vantagens da arquitectura Blender é o seu suporte à linguagem Python e a facilidade de integração de scripts desenvolvidos por utilizadores. Qualquer pessoa pode criar add-ons (extensões ou plugins) em Python para o Blender. Alguns add-ons são distribuídos com a aplicação, integram a distribuição oficial, mas existem muitos outros que podem ser integrados pelo próprio utilizador.

Instalar/ativar

Aceda ao menu File, opção User Preferences. Na janela de configuração, escolha o painel Add-Ons.

Neste painel, pode activar os Add-Ons que já estão instalados e organizados por categorias (i.e. Enabled, Disabled, Add Mesh, etc.) ou pode instalar novos add-ons através do botão Install Add-On.

Se quiser tornar permanente a instalação/ativação dos Add-Ons, para evitar ter de voltar a activar sempre que inicia o Blender, pode gravar a configuração com os add-ons activados como configuração pré-definida através do botão Save As Default.

A Wiki oficial é o melhor local para obter mais informações sobre os add-ons que são distribuídos com o Blender ou recolher novos add-ons. Para aceder diretamente à página de extensões em python para o Blender, clique aqui.

Alguns Add-Ons recomendados que não surgem nas listas anteriores

Edit Linked Library

Permite editar objetos linkados diretamente nos ficheiros originais sem ter de sair do Blender ou abrir nova instância.

Este addon já vem com o Blender, basta ativar.

Outros Add-Ons interessantes

Align Tools
Cria um painel novo na barra de ferramentas (Tools), visível em Object Mode, que disponibiliza diversos botões que facilitam o alinhamento da localização, rotação e escala entre vários objectos.
Copy Attributes Menu
Na janela de 3D View, passa a poder utilizar o atalho CTRL+C para copiar diversos atributos entre objectos. Por exemplo, se tiver vários objectos seleccionados e clicar em CTRL+C, pode copiar os valores relativos à rotação do objecto activo (o último objecto seleccionado) e aplicar os mesmos nos restantes objectos seleccionados.
Measure Panel
Cria um painel novo na barra de propriedades (N) onde surge a medição da distância entre um objecto e o cursor 3D ou entre dois objectos, caso tenha dois objectos seleccionados. O painel possibilita ainda a activação de uma opção que permite ver o valor da distância na própria janela de 3D View.
Import MakeHuman
Cria uma nova opção no menu de Import (File>Import>MakeHuman) que permite importar modelos criados no programa MakeHuman e gravados no seu formato próprio (.mhx). Deixa de ser necessário converter para outro formato, como COLLADA (.dae) ou Wavefront (.obj).
