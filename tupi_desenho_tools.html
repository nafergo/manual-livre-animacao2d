Ferramentas de desenho

Desenhar

sss

Pencil (P)
Permite traçar livremente linhas de tipo path (uma linha com pontos/nós que podem depois ser editados/manipulados). O parâmetro Smooth (barra de propriedades da ferramenta) define o grau de arredondamento/suavização da linha.
Ink (I)
Permite traçar livremente linhas de tinta de tipo path (uma linha com pontos/nós que podem depois ser editados/manipulados). O parâmetro Smooth (barra de propriedades da ferramenta) define o grau de arredondamento/suavização da linha.
PolyLine (S)
Permite traçar ponto-a-ponto linhas de tipo path (uma linha com pontos/nós que podem depois ser editados/manipulados). X ou BDR terminam a linha.
Line (L)
Permite traçar ponto-a-ponto linhas retas de tipo path (uma linha com pontos/nós que podem depois ser editados/manipulados). X ou BDR terminam a linha. Shift permite alinhar linha na vertical ou horizontal.
Square (R)
Permite traçar quadrado. Shift permite desenhar quadrado proporcional.
Elipse (C)
Permite traçar elipse. Shift permite desenhar elipse proporcional (círculo).

Seleção e edição de nós

Seleção de objeto (O)
Permite selecionar objeto. A barra de propriedades apresenta algumas ferramentas auxiliares: flip, ordenar (sobreposição, agrupar e posição (em X e Y).
Seleção de nós (N)
Permite selecionar nós de linhas.

Preenchimento

Preenchimento (I)
Pinta o interior de uma forma.
Contorno (B)
Pinta o contorno ou linha.

Propriedades do pincel

Permite definir vários caraterísticas do pincel (grossura, padrão, etc.).
